# Sistem de Acces - Salt Edge Test

Acesta este un proiect demonstrativ scris în **Ruby** pentru gestionarea accesului angajaților.

## 🚀 Funcționalități

- Verificarea accesului pe bază de nume (cu formatare automată).
- Încărcarea bazei de date dintr-un fișier extern (`.txt`).
- Logica bazată pe Programare Orientată pe Obiecte (OOP).
- Gestionarea erorilor (Error Handling) pentru fișiere lipsă.

## 🛠️ Tehnologii folosite

- **Ruby 3.4.x**
- **Git & GitHub**
- **VS Code**

## 🏁 Cum se rulează

```bash
ruby sistem_acces.rb
```
